Необходимо реализовать приложение со следующим функционалом:

В коде можно создать любое количество объектов представляющих информацию о многоугольниках. Каждый объект должен отображаться на HTML канвасе в виде 2D фигуры. Канвас имеет размеры экрана с отступами от края экрана.  При первом запуске приложение отображает созданные фигуры списком, одну под другой, на небольшом расстоянии друг от друга.

Пользователь может передвигать элементы по канвасу с помощью механизма drag & drop, при этом с элементом можно взаимодействовать только если зажать кнопку мыши в пределах границ фигуры.

Объект обновляет свое состояние после drop'а. Если после drop'а два объекта пересекаются они должны поменять цвет заливки фигуры на красный.

Приложение должно быть реализовано без применения готовых библиотек, реализующих логику/функциональные требования описанные выше. Вся логика взаимодействия с канвасом должна использовать только стандартное API JavaScript'а.
